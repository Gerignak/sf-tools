Options

expr Overrides reserved column name

This option specifies an expression that will be used for calculating the cell's value.

format Overrides extra option

This option specifies an expression that will be used for determing what will be displayed in the cell.
It does not affect sorting in any way.

brackets Global Shared

Use this option to set whether you want brackets around the difference in the cell.

debug Global

Show limited debug information in the browser's console.

members Global

Show class structure of the guild and members that left or joined in the selected timespan.

outdated Global

Highlight players that are not the most recent. This option is enabled by default.

performance Global

Limits the number of displayed players to specified value.
This option applies only to the Players tab and should be used if you are working with larger amount of players (above 1000).

alias

Changes the name of the column.

width Shared

Changes the width of the column.

extra

Appends a string at the end of the cell content. It does not apply if you are using format option.

difference Global Shared

This options allows the cells to show difference against selected timespan.

statistics Global Shared

This option shows a smallmum, average and maximum values in the column. This has effect only within Guilds tab.

percentage Global Shared

This option does the same thing as extra % option would do, but works only for few selected reserved column names.

hydra Global Shared

This option shows whether player has the Dehydration achievement, but works only for Awards column.

grail Global Shared

This option shows whether player has the Holy Grail unlocked, but works only for Album column.

visible Global Shared

This option determines, if the cell content should be shown or not (it will be drawn with the highlight color, if any is used).

maximum Global Shared

This option shows the maximum knights player can currently have, but works only for Knights column.

flip Global Shared

This option will treat lower values as better when sorting.

color

This option will create a color rule for cell. The first that matches current value will be used.
In case that no color rules match, the default will be used.

value Overrides format option

This option will create a value rule for cell. The first that matches current value will be used.
In case that no color rules match, the default will be used.
